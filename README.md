# NEST BACKEND PROJECT
Масштабируемый бэкенд на Nwst.js (создание пользователей, выделение ролей, ограничение доступа, создание постов, хранение файлов в файловой системе). В качестве СУБД используется PostgreSQL в связке с ORM Sequelize на node js. API Задокументирован с помощью SWAGER. Приложение обернуто в docker.

В режиме dev сервер крутится на 5001 порту. 
SWAGER - http://localhost:5001/api/docs#/
